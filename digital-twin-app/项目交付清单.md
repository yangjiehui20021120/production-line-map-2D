## 📋 部署要求

### 环境要求
- **Node.js**: 16.0+ 
- **npm**: 7.0+
- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+

### 部署步骤
1. 解压项目包
2. 安装依赖 (`npm install`)
3. 启动开发服务器 (`npm run dev`)
4. 构建生产版本 (`npm run build`)

### 访问地址
- **开发环境**: http://localhost:5173
- **生产环境**: 根据部署方式确定
